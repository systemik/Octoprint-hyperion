# Octoprint-hyperion
Drive hyperion lights via MXXX commands

Usefull command args are -e ; -x ; -c


Hyperion command line :

hyperion-remote:

Usage: hyperion-remote [OPTIONS]

Parameters:
  -c, --color <arg>
  -e, --effect <arg>
  -x, --clear
      --clearall
